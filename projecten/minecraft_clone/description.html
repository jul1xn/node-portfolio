Better MC is een persoonlijke remake van een sandbox game, ontwikkeld als eigen project. In deze versie kun je een 3D wereld bouwen en verkennen, blokken plaatsen en verwijderen, en je eigen kleine wereld creëren zoals in een bekende game. Het doel was om de basis-mechanica van een open wereld te leren en zelf uit te werken.

Voor dit project heb ik Unity gebruikt om de game-engine te benutten, samen met programmeren in C# voor de gameplay scripts. Ik heb bijvoorbeeld de blokkenlogica, camerabeweging, en collision rules zelf opgezet en getest. Het project is beschikbaar op GitHub zodat iedereen de code kan bekijken, aanpassen of uitbreiden.

Daarnaast heb ik het project verder uitgebreid met eigen ideeën, zoals een eenvoudige inventory en verschillende bloktypes, zodat het bouwen en verkennen leuker wordt. Door zelf te experimenteren met scripts en de Unity-engine heb ik veel geleerd over 3D-logica, physics en performance optimalisatie. Het project staat open op GitHub, zodat anderen kunnen zien hoe het in elkaar zit, er inspiratie uit kunnen halen of zelf functies kunnen toevoegen.
