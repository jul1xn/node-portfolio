Dit project is een Windows Forms applicatie die ik heb gemaakt als schoolproject. De app maakt het mogelijk om een kenteken in te voeren en vervolgens alle belangrijke voertuiggegevens op te vragen via de RDW API. De gebruiker kan daarna door verschillende tabs bladeren om algemene informatie, technische gegevens en eventuele gebreken van het voertuig te bekijken. Dit maakt de app handig voor iedereen die snel iets wil controleren voordat hij of zij een auto koopt.

De applicatie is gebouwd in Visual Studio Community 2022 met .NET Windows Forms. Voor de structuur en het ontwerp heb ik eerst wireframes gemaakt in Adobe XD. Daarna heb ik de functionaliteiten stap voor stap ontwikkeld, zoals het valideren van kentekens, het ophalen van voertuigdata via de RDW API en het tonen van een foto van het voertuig via de Google Images API.

Tijdens het project heb ik meerdere testrondes uitgevoerd om te controleren of alle onderdelen goed werkten, zoals foutmeldingen bij verkeerde invoer en de schaalbaarheid van de UI. De feedback van testers heb ik gebruikt om de app duidelijker en gebruiksvriendelijker te maken. Het complete project, inclusief code, is terug te vinden op mijn GitHub.
