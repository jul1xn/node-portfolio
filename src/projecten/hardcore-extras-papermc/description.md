Dit project heet **Hardcore Extras** en is mijn eerste Minecraft Java plugin. Het is gemaakt om extra opties aan te bieden voor Minecraft servers in een hardcore-stijl spelmodus. Met deze plugin kun je configureren wat er gebeurt als een speler sterft, zoals het aantal levens dat ze hebben of hoe lang ze tijdelijk geband worden. Het doel is om server eigenaren meer controle te geven over hoe sterven en voortgang werkt in hun serveromgeving.

De plugin voegt een aantal systemen toe, zoals een levensysteem waarbij spelers een bepaald aantal levens krijgen en een deathban systeem dat spelers tijdelijk kan verbannen bij een dood. Er zit ook een grace-periode in: een tijd waarin je geen penalty krijgt als je net begonnen bent in de server. Je kunt als speler commands gebruiken om je resterende levens of grace-tijd te controleren.

Ik heb deze plugin gemaakt met **Java** en de **Bukkit/Spigot API**. De plugin is gebouwd voor **PaperMC** en **Folia** server versies, die bekend staan om hun performance en moderne serverstructuur. Tijdens het project heb ik geleerd hoe je events afhandelt, commands maakt en instellingen configureerbaar maakt voor server eigenaren. De code staat op GitHub en de plugin is ook te downloaden via Modrinth om zelf uit te proberen.
